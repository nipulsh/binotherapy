# Image Requirements for Shadow Pursuit Game

The following images need to be added to the `/public/` folder:

1. **thief.png** - A simple PNG image of a thief icon/character (blue dot replacement)
   - Recommended size: 64x64 pixels or larger
   - Should be a simple icon or character sprite
   - Transparent background preferred

2. **police.png** - A simple PNG image of a police officer icon/character (black dot replacement)
   - Recommended size: 64x64 pixels or larger
   - Should be a simple icon or character sprite
   - Transparent background preferred

3. **star.png** - A simple PNG image of a star icon (for scoring system)
   - Recommended size: 64x64 pixels or larger
   - Should be a simple star icon
   - Transparent background preferred
   - Gold/yellow color recommended

## Where to find images:
- Free icon sites: Flaticon, Icons8, Freepik
- Search terms: "thief icon png", "police officer icon png", "star icon png"
- Make sure images are free to use or have appropriate licenses

## Once images are added:
The game will automatically load and use these images. No code changes needed.
